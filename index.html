O que é corrida?

Corrida é um esporte que envolve correr em uma pista ou em um campo aberto, com o objetivo de completar uma distância específica no menor tempo possível.

ATLETISMO

O que é salto em altura?

O salto em altura é uma modalidade de atletismo onde o atleta deve saltar sobre uma barra horizontal, utilizando técnicas específicas para superar a altura desejada.

ATLETISMO

O que é arremesso de peso?

O arremesso de peso é uma modalidade do atletismo no qual o arremessador deve jogar uma bola entre 1 a 5 kg o mais longe possível.
